# SkillSphere

SkillSphere is a full-stack Student Skill Exchange Platform that enables students to exchange knowledge instead of money.

## Features
- AI Skill Matching
- Student Profiles
- Real-time Chat
- Learning Sessions
- Gamification
- Progress Tracking
- Dashboard
- Admin Panel

## Tech Stack
- React
- TypeScript
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Installation

npm install
npm run dev
